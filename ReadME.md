# SilentCPU - Based on Lime Miner v0.3


## Main Features

* .NET - Coded in Visual Basic .NET, requires .NET Framework 4.0.

* Injection - Hide payload behind another process

* CPU & GPU Mining - Can mine on Both CPU and GPU (Nvidia & AMD)
